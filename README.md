# Chronicles of the Mind: Genesis Pitch

**Artist:** Mengzhen Yang  
**Submission:** D&AD Awards — Presentation Images

## Quick Start

1. **Preview** — Open `pitch-deck.html` in any browser
2. **Export JPGs** — Run `npm install && npm run export`

## Structure

| Page | Content |

| Cover | MENGZHEN YANG — Title |

| 14 | The Ask — Institutional Requirements |
| Back | Colophon |

## Export Specs

- Resolution: 3840×2160 (2× retina)
- Format: JPG, quality 95
- Meets D&AD minimum: 1200px+, <50MB per file

## Adding Your Artwork

Replace placeholder visuals with your high-res artwork:

1. Place images in `assets/` folder
2. Update `<img src="...">` paths in `pitch-deck.html`
3. Re-export with `npm run export`
